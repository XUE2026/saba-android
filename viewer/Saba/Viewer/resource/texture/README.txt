This directory is reserved for environment textures used by the EnvironmentManager.

Expected texture files:
  - grass.png       : Grass ground texture
  - mud.png         : Mud ground texture
  - asphalt.png     : Asphalt ground texture
  - field.png       : Field ground texture
  - tree.png        : Tree billboard sprite
  - flower.png      : Flower billboard sprite
  - bird.png        : Bird billboard sprite
  - butterfly.png   : Butterfly billboard sprite

Place PNG texture files here to enable textured rendering.
When textures are absent, the EnvironmentManager will use solid colors as fallback.
