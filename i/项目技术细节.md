# 项目技术细节

## 技术实现要点

### ModelCache
- 使用 `std::vector<CacheEntry>` 存储缓存条目
- `std::unordered_map<std::string, size_t>` 实现 O(1) 路径查找
- LRU 使用 `std::chrono::steady_clock::time_point` 记录访问时间
- 淘汰时使用 `std::min_element` 查找最久未访问的条目
- 支持 PMX 和 PMD 两种模型格式

### GLTextureCache
- 组合键：`path + ":" + genMipMap + ":" + rgba` 支持不同加载参数
- 使用 `glGetTexLevelParameteriv` 获取纹理尺寸和内部分辨率
- 内存估算：width * height * bytesPerPixel * (4/3 if mipmap)

### TextureCompression
- `GL_COMPRESSED_RGB8_ETC2 (0x9274)` 和 `GL_COMPRESSED_RGBA8_ETC2_EAC (0x9278)`
- 先普通加载，再通过 `glTexImage2D` 重新指定压缩格式触发 GPU 压缩
- ConvertToETC2 为软件回退实现

### ModelPreloader
- 使用 `std::async(std::launch::async, ...)` 创建后台任务
- `std::atomic<float>` 保证进度读取线程安全
- 支持 `CancelPreload()` 通过 `std::atomic<bool>` 标志

### OptimizationPanel
- 使用 ImGui 标准控件（Slider, Button, ProgressBar, Checkbox）
- "Optimize for Mobile" 一键切换缓存大小
- 展示当前缓存使用情况
