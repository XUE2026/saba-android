# 项目说明

## Saba MMD Viewer 资源优化模块

该项目为 Saba MMD Viewer 实现了资源优化功能，主要针对 Android 平台优化模型加载速度和内存效率。

### 核心模块

1. **ModelCache** - 模型缓存系统
   - 单例模式，全局访问
   - LRU 淘汰策略
   - 线程安全
   - 支持加载 PMX/PMD 格式

2. **GLTextureCache** - OpenGL 纹理缓存
   - GPU 纹理缓存，避免重复加载
   - 自动内存估算
   - LRU 淘汰

3. **ModelPreloader** - 模型预加载器
   - 异步后台加载
   - 进度回调
   - 取消支持

4. **OptimizationPanel** - ImGui 优化面板
   - 缓存大小配置
   - 预加载控制
   - 内存监控
   - 一键移动优化
