# 操作记录

## 任务：Saba MMD Viewer 资源优化实现

### 创建的文件

1. **ModelCache.h / ModelCache.cpp** (src/Saba/Model/MMD/)
   - 单例模型缓存系统
   - LRU 淘汰策略，默认最大缓存 5 个模型
   - 线程安全（std::mutex）
   - 支持 PMX 和 PMD 模型格式

2. **GLTextureCache.h / GLTextureCache.cpp** (viewer/Saba/GL/)
   - 单例纹理缓存系统
   - LRU 淘汰策略，默认最大缓存 50 个纹理
   - 线程安全
   - 支持内存使用估算

3. **ModelPreloader.h / ModelPreloader.cpp** (viewer/Saba/Viewer/)
   - 后台线程模型预加载
   - 使用 std::async
   - 支持进度回调

4. **OptimizationPanel.h / OptimizationPanel.cpp** (viewer/Saba/Viewer/)
   - ImGui 优化设置面板
   - 缓存设置控制
   - 预加载器控制
   - 内存使用显示
   - 移动端优化开关

### 更新的文件

5. **GLTextureUtil.h / GLTextureUtil.cpp** - 添加纹理压缩支持
   - LoadTextureCompressed() - 加载时请求 GPU 压缩
   - ConvertToETC2() - 软件回退的 ETC2 转换

6. **src/CMakeLists.txt** - 添加 ModelCache 文件
7. **viewer/CMakeLists.txt** - 添加 GLTextureCache、ModelPreloader、OptimizationPanel 文件
