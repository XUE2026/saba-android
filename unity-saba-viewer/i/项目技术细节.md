# 项目技术细节

## 命名空间
所有脚本使用 `UnitySabaViewer` 命名空间。

## 脚本依赖关系

### ModelLoader
- **BoneController** - 初始化骨骼系统
- 使用 `Task.Run` 异步加载文件数据
- 维护 `Dictionary<string, GameObject>` 缓存
- 支持 PMX/PMD 格式

### CameraController
- 独立组件，不依赖其他脚本
- 使用 LateUpdate 平滑插值
- 支持触控(单指拖拽/双指缩放)和鼠标输入
- 暴露 distance, rotationX, rotationY, target 属性

### FilterManager
- 使用 Unity PostProcessing Volume 的 ColorGrading 效果
- 6个预设: WhiteBright, SkinBright, Dark, GenshinDay, GenshinNight, GenshinDusk
- 每个预设控制 exposure, contrast, saturation, hueShift, temperature, tint
- 按索引或名称应用预设

### EnvironmentManager
- 控制 Directional Light
- 5种地面类型: Grid/Checker/Grass/Dirt/Asphalt
- 4种基本体: Cube/Sphere/Cone/Cylinder
- 程序化生成网格和棋盘格纹理

### BoneController
- `Dictionary<string, Transform>` 管理骨骼
- 保存骨骼默认位置/旋转用于重置
- 提供骨骼列表获取、旋转、位移、重置方法

### AnimationController
- 4种内置动画: Idle, Walk, Run, Jump
- 使用 BoneKeyframe 结构体定义骨骼关键帧
- 支持 CrossFade 动画过渡
- 覆盖 MMD 标准骨骼名称

### UIManager
- 管理 Canvas 和 8个 UI 面板
- 提供按钮/滑动条/下拉菜单的访问方法
- 面板显隐控制

### LanguageManager
- 单例模式, DontDestroyOnLoad
- 3种语言: 中文, 英文, 日文
- 50+ UI 字符串键
- 语言偏好保存到 PlayerPrefs

### RecordingManager
- 以 30fps 录制骨骼变换
- RecordedFrame 包含时间戳、相机姿态、骨骼姿态
- 导出 VMD (二进制格式)
- 导出 GoBotAction (JSON 格式)

### GobotIntegration
- 加载/播放 .gobot_action 文件
- 加载 .gobot_scene 文件(环境设置)
- 帧间线性插值播放
- 导出 .gobot_project

### BuildHelper
- 编辑器菜单工具
- Release/Development 两种构建模式
- 自动设置 IL2CPP, ARM64, API Level
- 构建缓存清理功能
