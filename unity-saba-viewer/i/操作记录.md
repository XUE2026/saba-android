# 操作记录

## 2026-05-31

### 创建 Unity SabaViewer 项目

创建了一个完整的 Unity 2022.3 LTS 项目，用于 MMD 3D 模型查看器（Android 目标平台）。

#### 创建的文件:

**项目设置文件:**
1. ProjectSettings/ProjectVersion.txt - Unity 版本 2022.3.0f1
2. Packages/manifest.json - 包含 Unity 2022.3 兼容包（PostProcessing, URP, UI 等）
3. ProjectSettings/ProjectSettings.asset - 项目设置（Android 目标, 质量设置等）

**C# 脚本 (11个):**
1. Assets/Scripts/ModelLoader.cs - PMX/PMD 模型加载器（异步, 缓存）
2. Assets/Scripts/CameraController.cs - 轨道相机控制（触控拖拽, 双指缩放）
3. Assets/Scripts/FilterManager.cs - 6种滤镜预设（后处理效果）
4. Assets/Scripts/EnvironmentManager.cs - 场景环境控制（太阳光, 地面, 基本体）
5. Assets/Scripts/BoneController.cs - 骨骼操控系统
6. Assets/Scripts/AnimationController.cs - 内置 MMD 风格动画（Idle/Walk/Run/Jump）
7. Assets/Scripts/UIManager.cs - 主 UI 画布面板管理
8. Assets/Scripts/LanguageManager.cs - 多语言支持（中/英/日, 50+ 字符串键）
9. Assets/Scripts/RecordingManager.cs - 骨骼动画录制与导出（VMD/GoBotAction）
10. Assets/Scripts/GobotIntegration.cs - GoBot 格式文件集成
11. Assets/Editor/BuildHelper.cs - Android APK 自动构建工具
