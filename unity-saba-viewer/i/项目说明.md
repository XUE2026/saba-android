# SabaViewer - MMD 3D 模型查看器

## 项目概述
一个基于 Unity 2022.3 LTS 的 Android 平台 MMD 模型查看器应用。

## 核心功能
- PMX/PMD 模型加载与显示
- 轨道相机控制（触控/鼠标）
- 后处理滤镜效果（6种预设）
- 场景环境控制（光照/地面/基本体）
- 骨骼操控系统
- 内置 MMD 风格动画
- 动画录制与导出（VMD/GoBotAction）
- GoBot 格式集成
- 多语言支持（中文/英文/日文）

## 技术栈
- Unity 2022.3.0f1
- Universal Render Pipeline (URP)
- Unity PostProcessing
- IL2CPP 脚本后端
- 目标: Android 64-bit (ARM64)

## 项目结构
```
Assets/
├── Scripts/          # C# 业务逻辑脚本
```
